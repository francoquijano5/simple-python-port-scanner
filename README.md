Simple Port Scanner in Python
Este proyecto es un escáner de red básico que verifica la disponibilidad de puertos específicos en una dirección IP utilizando el protocolo TCP.
   
   Características:
* Escaneo de puertos específicos definidos por el usuario.
* Manejo de timeouts para optimizar la velocidad.
* Uso de la librería nativa socket de Python.

  Cómo usarlo:
1. Clona el repositorio:
git clone https://github.com/TU_USUARIO/simple-python-port-scanner.git

2. Ejecuta el script:
python scanner.py

Requisitos
* Python 3.x instalado.
